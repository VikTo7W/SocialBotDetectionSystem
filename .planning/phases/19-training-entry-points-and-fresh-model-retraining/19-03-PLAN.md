# Plan 19-03

## Objective

Build the maintained TwiBot training entry point and align the TwiBot native path with the new artifact naming contract.

## Tasks

1. Introduce `train_twibot.py` as the maintained TwiBot training command over the shared `CascadePipeline("twibot")` core.
2. Move the maintained default artifact to `trained_system_twibot.joblib` and preserve `train_twibot20.py` only as a compatibility shim if needed.
3. Update native-evaluation defaults and tests so the TwiBot-native path reads the renamed maintained artifact cleanly.

## Verification

- targeted pytest for TwiBot training/evaluation entry-point tests

- Running `train_twibot.py` is the maintained path for TwiBot training.

- Native TwiBot evaluation remains functional against the maintained artifact contract.
