<decisions_made>

- **Optuna TPESampler** chosen over scikit-optimize (Python 3.13 incompatible) and hyperopt for threshold calibration
- **calibrate.py as standalone file** — does not modify botdetector_pipeline.py routing logic; only system.th is mutated in-place
- **s2a_bot lower bound = max(s2a_human + 0.05, 0.70)** — prevents human/bot threshold inversion in Bayesian search
- **evaluate.py returns dict AND prints** — dict enables programmatic testing; print produces paper-ready console output
- **Stage 3 graph for API: empty edges DataFrame + nodes_total=1** — graceful degradation for single-account inference without graph context
- **Eager module-level joblib.load in api.py** — Starlette TestClient doesn't trigger async lifespan; eager load on module reload makes tests work while lifespan still handles production startup
- **predict_system calling convention bug patched in api.py** — `bp.extract_stage1_matrix` and `bp.extract_stage2_features` replaced at module level; source not modified

</decisions_made>
