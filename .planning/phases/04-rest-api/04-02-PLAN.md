**Pydantic models:**
- `MessageItem(BaseModel)`: text (Optional[str] = None), ts (Optional[float] = None), model_config = {"extra": "allow"}
- `AccountRequest(BaseModel)`: account_id (str, REQUIRED), username (str, REQUIRED), submission_num (float = 0.0), comment_num_1 (float = 0.0), comment_num_2 (float = 0.0), subreddit_list (List[str] = Field(default_factory=list)), profile (Optional[str] = ""), messages (List[MessageItem] = Field(default_factory=list))
- `PredictResponse(BaseModel)`: p_final (float), label (int)

**Adapter function** `_to_dataframe(req: AccountRequest) -> pd.DataFrame`:
- Converts MessageItem list to list of dicts via `[m.model_dump() for m in req.messages]`
- Creates single-row DataFrame with columns: account_id, node_idx (np.int32(0)), label (0, dummy), username, profile, subreddit_list, submission_num, comment_num_1, comment_num_2, messages
- node_idx=0 and label=0 are fixed for single-account inference

**Route** — use `def predict(...)` (SYNC, not async) to avoid blocking the event loop with CPU-bound inference:
```python
@app.post("/predict", response_model=PredictResponse)
def predict(req: AccountRequest, request: Request):
    df = _to_dataframe(req)
    try:
        result = predict_system(
            sys=request.app.state.system,
            df=df,
            edges_df=EMPTY_EDGES,
            nodes_total=1,
        )
    except (ValueError, RuntimeError) as exc:
        raise HTTPException(status_code=422, detail=str(exc))
    p_final = float(result["p_final"].iloc[0])
    return PredictResponse(p_final=p_final, label=int(p_final >= 0.5))
```

Key decisions already locked:
- Sync route (not async) per user decision — prevents event loop blocking
- MODEL_PATH from env var, default "trained_system.joblib"
- Empty edges DataFrame for Stage 3 graph — single account has no graph context
- nodes_total=1 — single node for single account inference
- Pydantic auto-422 on missing required fields (account_id, username)
- try/except around predict_system for domain errors -> 422
