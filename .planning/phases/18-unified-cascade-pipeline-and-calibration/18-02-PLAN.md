# Plan 18-02

## Objective

Build the shared dataset-aware cascade pipeline layer and make it the maintained source of truth for training and prediction orchestration.

## Tasks

1. Introduce a dedicated shared pipeline module, likely `cascade_pipeline.py`, with class-based orchestration around fitting and prediction.

3. Reduce `botdetector_pipeline.py` to stage-model definitions plus thin compatibility wrappers or re-exports where needed.

## Acceptance Criteria

- One shared implementation handles both BotSim and TwiBot cascade orchestration.
- The maintained source of truth is class-based and dataset-aware.
- No duplicated training/prediction orchestration remains across maintained code paths.
