   - Patches the MODEL_PATH used by api.py: `monkeypatch.setenv("MODEL_PATH", str(model_path))`

   ```python
   }
   ```

3. Five test functions:

   a. `test_predict_returns_200(client)` — POST /predict with VALID_PAYLOAD; assert status_code == 200; assert response JSON has keys "p_final" and "label".

   b. `test_predict_output_range(client)` — POST /predict with VALID_PAYLOAD; assert 0.0 <= data["p_final"] <= 1.0; assert data["label"] in (0, 1); assert data["label"] == int(data["p_final"] >= 0.5).

   c. `test_startup_loads_system(client)` — Assert client.app.state.system is not None; assert hasattr(client.app.state.system, "stage1").
