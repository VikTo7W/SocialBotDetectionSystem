# Milestone v1.3: Twibot System Version

**Status:** SHIPPED 2026-04-18
**Phases:** 11-13
**Total Plans:** 6

## Overview

Turned the TwiBot-20 zero-shot transfer path (shipped in v1.2) into a reproducible, evidence-backed system version with fresh live evaluation artifacts, a hardened execution flow, and explicit release documentation. No model retraining was performed — the v1.2 `trained_system_v12.joblib` cascade was reused unchanged.

## Phases

**Depends on:** Phase 10
**Requirements:** REPRO-01, REPRO-02, REPRO-03
**Plans:** 2 plans

Plans:
- [x] 11-01-PLAN.md — Fix two failing test stubs in tests/test_evaluate_twibot20.py so Phase 9 kwargs forwarding is absorbed by monkeypatched lambdas (REPRO-02, REPRO-03)

## Milestone Summary

**Key Decisions:**

- Windows temp friction is pytest-level, not production — zero `tempfile`/`gettempdir` usage confirmed in production code; pytest-level friction acknowledged rather than worked around
- `output_dir` parameter added to `evaluate_twibot20.py __main__` — backward-compatible, os.makedirs-safe, all three artifact writes route through os.path.join
- TwiBot zero-shot F1=0.0 is the documented transfer regime for v1.3 — not treated as a pipeline bug

**Issues Resolved:**

- Fragile default temp/cache artifact path replaced with explicit output_dir routing
- Test stub incompatibility with Phase 9 kwargs forwarding fixed
- Table 5 and cross-dataset paper outputs now read from and write to stable milestone-owned paths

**Issues Deferred:**

- Full pytest/runtime verification in a cleaner (non-Windows-permission-constrained) environment
- No dedicated v1.3 milestone audit was run before close
- Stale root-level TwiBot artifacts from pre-Phase-12 runs remain in the repo root

**Technical Debt Incurred:**

- Requirements traceability table remained unchecked throughout execution (all requirements are delivered but the table was not updated during phases)
- ROADMAP.md and STATE.md required manual correction rather than gsd-tools automation (non-standard workstreams path layout)
