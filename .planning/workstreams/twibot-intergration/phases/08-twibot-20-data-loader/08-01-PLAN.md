Function `load_accounts(path: str) -> pd.DataFrame`:

1. Open `path` with `encoding="utf-8"`, `json.load(f)` into `data` list.
2. Initialize `global _no_neighbor_count` and set to 0.
3. Iterate `for idx, record in enumerate(data)`:
   - `profile = record["profile"]`
   - `tweets = record.get("tweet") or []` (Pitfall 2 guard: tweet can be None)
   - Build messages list comprehension:
     `[{"text": str(t), "ts": None, "kind": "tweet"} for t in tweets if t]`
     (per D-01 — each tweet is a schema-compatible dict; skip falsy tweets)
   - Count no-neighbor: `if record.get("neighbor") is None: _no_neighbor_count += 1`
   - Append row dict:
     ```python
     {
         "node_idx": np.int32(idx),
         "screen_name": str(profile.get("screen_name", "") or "").strip(),
         "statuses_count": int(str(profile.get("statuses_count", 0) or 0).strip() or 0),
         "followers_count": int(str(profile.get("followers_count", 0) or 0).strip() or 0),
         "friends_count": int(str(profile.get("friends_count", 0) or 0).strip() or 0),
         "created_at": str(profile.get("created_at", "") or "").strip(),
         "messages": messages,
         "label": int(record["label"]),
     }
     ```
   - The `int(str(val).strip() or 0)` pattern handles: None values, string values with trailing whitespace, and empty strings (Pitfall 1, Pitfall 6).
4. After loop: `df = pd.DataFrame(rows)`, then `df["node_idx"] = df["node_idx"].astype(np.int32)`.
5. Return `df`.

The function must NOT sort messages by ts (unlike botsim24_io.py — all ts are None for TwiBot-20).
The function must NOT parse created_at (per D-02 — store as raw string).
