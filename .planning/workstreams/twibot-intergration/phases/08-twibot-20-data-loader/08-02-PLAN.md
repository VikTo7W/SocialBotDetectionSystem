Add two functions to `twibot20_io.py` after the existing `load_accounts()` function.

**Function 1: `build_edges(accounts_df: pd.DataFrame, path: str) -> pd.DataFrame`**

1. Open `path` with `encoding="utf-8"`, `json.load(f)` into `data`.
2. Build ID-to-node_idx lookup: `id_to_idx = {record["ID"]: int(accounts_df.iloc[i]["node_idx"]) for i, record in enumerate(data)}`.
3. Precompute `WEIGHT = np.float32(np.log1p(1.0))`.
4. Initialize `rows = []` (list of tuples).
5. Iterate `for i, record in enumerate(data)`:
   - `neighbor = record.get("neighbor")`
   - `if neighbor is None: continue`
   - `src_idx = int(accounts_df.iloc[i]["node_idx"])`
   - For following edges (etype=0): `for nid in (neighbor.get("following") or []):` — if `nid in id_to_idx`, append `(src_idx, id_to_idx[nid], 0, WEIGHT)`. Direction: src=current account, dst=neighbor (D-05, Pitfall 4).
   - For follower edges (etype=1): `for nid in (neighbor.get("follower") or []):` — if `nid in id_to_idx`, append `(id_to_idx[nid], src_idx, 1, WEIGHT)`. Direction: src=neighbor, dst=current account (D-05, Pitfall 4).
6. After loop, unzip: `if rows: srcs, dsts, etypes, weights = zip(*rows)` else `srcs, dsts, etypes, weights = [], [], [], []` (Pitfall 5 — handle empty case).
7. Return `pd.DataFrame({"src": np.array(srcs, dtype=np.int32), "dst": np.array(dsts, dtype=np.int32), "etype": np.array(etypes, dtype=np.int8), "weight": np.array(weights, dtype=np.float32)})`.

**Function 2: `validate(accounts_df: pd.DataFrame, edges_df: pd.DataFrame) -> None`**

1. Define `required_cols = ["node_idx", "screen_name", "statuses_count", "followers_count", "friends_count", "created_at", "messages", "label"]`.
2. `missing = [c for c in required_cols if c not in accounts_df.columns]`
3. `assert not missing, f"Missing columns in accounts_df: {missing}"`
4. `n = len(accounts_df)`
5. Guard for empty edges: `if len(edges_df) > 0:` (Pitfall 5)
   - `assert int(edges_df["src"].max()) < n, f"src index {int(edges_df['src'].max())} out of bounds (n={n})"`
   - `assert int(edges_df["dst"].max()) < n, f"dst index {int(edges_df['dst'].max())} out of bounds (n={n})"`
6. Compute no-tweet fraction: `no_tweet_frac = accounts_df["messages"].map(len).eq(0).mean()`
7. Read module-level `_no_neighbor_count` (set by `load_accounts`): `global _no_neighbor_count`
8. Print diagnostics per D-03 (print, not logging):
   ```python
   print(f"[twibot20] accounts: {n}, edges: {len(edges_df)}")
   print(f"[twibot20] no-neighbor fraction: {_no_neighbor_count / n:.3f}" if n > 0 else "[twibot20] no accounts")
   print(f"[twibot20] no-tweet fraction: {no_tweet_frac:.3f}")
   ```
